# NodeJS - Send Email - Gửi Email
- NodeJS
- ExpressJS
- Nodemailer

## Yêu Cầu

* NodeJS 8.10.x

# RUN

```
$ npm install

--- Chạy Server ---

-------- Cách 1 --------
$ node src/server.js

-------- Cách 2 --------
$ cd src
$ node server.js
```

### Thay đổi tài khoản email trong file: **src/utils/mailer.js**
